# 字体创建失败



## 遇到错误



```txt
ERROR 10261 [http-nio-9000-exec-5] com.xht.auth.captcha.handler.AbstractCaptcha : 字体创建失败:Problem reading font data.

java.io.IOException: Problem reading font data.
```



## 解决方案



```shell
yum install fontconfig
```
